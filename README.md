# Data Cleaning and Visualization Project
 Project Overview

This project demonstrates the process of cleaning a raw dataset and creating meaningful visualizations using Python. The project includes handling missing values, removing duplicate records, detecting and removing outliers, and generating multiple graphs for data analysis.

Objective
The objective of this project is to:
- Load a dataset using Pandas.
- Clean the dataset by handling missing values and duplicate records.
- Detect and remove outliers using the IQR (Interquartile Range) method.
- Visualize the cleaned data using Matplotlib and Seaborn.
- Export the cleaned dataset for further analysis.


 Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

 Features

- Import dataset using Pandas
- Handle missing values
- Remove duplicate records
- Detect and remove outliers using IQR
- Generate multiple data visualizations
- Export cleaned dataset

 Visualizations Generated
The project generates the following visualizations:
1. Salary Distribution Histogram
2. Department Count Bar Chart
3. Age vs Salary Scatter Plot
4. Salary Box Plot
5. Correlation Heatmap
